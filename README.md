# FileUniquefy 微信文件去重工具

扫描微信文件目录下所有形如 `YYYY-MM` 的子文件夹，自动识别重复文件，将重复文件移动到上层的「重复」文件夹中，保持原始目录结构。

## 功能描述

- **自动发现**：启动后自动扫描系统中的微信文件目录，支持新旧两版微信路径
- **概览分析**：选定目录后立即展示各月份的文件数、大小疑似重复数、MD5 精确重复数
- **MD5 检测**：通过计算文件 MD5 哈希值精确判断文件是否重复
- **只处理一层**：仅扫描 `YYYY-MM` 下的直接文件，不递归进入子文件夹
- **保留最旧**：重复文件中保留修改时间最早的（原始文件），其余移动到「重复」目录
- **结构保持**：移动后的文件保持原始的目录层级结构
- **四阶段流程**：选目录 → 概览确认 → 移动计划 → 执行

## 支持的微信目录格式

| 版本 | 路径格式 |
|------|---------|
| 新版微信 | `xwechat_files/<账号>/msg/file/` |
| 旧版微信 | `WeChat Files/<账号>/FileStorage/File/` |

脚本会自动扫描所有盘符（Windows）或用户目录（macOS/Linux）下的常见位置，包括自定义的 Documents 路径。

## 安全防护

- 禁止对根目录、用户主目录、系统关键目录操作
- 路径层级过浅时要求确认
- 非微信路径要求额外确认
- 待移动文件数量超过 500 时额外警告

      (15 个月份文件夹)
  [2] D:\document\WeChat Files\bachopin\FileStorage\File
      (23 个月份文件夹)

  [0] 手动输入其他路径

请选择目录编号: 2

找到 23 个月份文件夹，正在分析重复情况...

  月份           文件数   大小疑似   MD5重复
  ------------ ------ -------- --------
  2024-04          32        8        5
  2024-05          18        -        -

确认后展示每个文件的保留/移动详情，再次确认才执行。

## 目录结构示例

```
WeChat Files/xxx/FileStorage/File/
├── 2025-11/
│   ├── report.pdf
│   └── photo.jpg
├── 2025-12/
│   ├── a.pdf          ← 保留（最新）
│   ├── a(1).pdf       ← 重复，将被移动
│   └── subdir/        ← 子文件夹，不处理
└── 重复/                ← 自动创建
    └── 2025-12/
        └── a(1).pdf   ← 移动到这里
```

## 使用方法

```bash
# 自动扫描微信目录，交互式选择
python FileUniquefy.py

# 通过命令行参数直接指定目录
python FileUniquefy.py "D:\document\WeChat Files\bachopin\FileStorage\File"
```
